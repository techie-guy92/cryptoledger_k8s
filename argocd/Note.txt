Pods that must be deleted after applying:

kubectl delete deployment argocd-applicationset-controller -n argocd
kubectl delete service argocd-applicationset-controller -n argocd
kubectl delete serviceaccount argocd-applicationset-controller -n argocd
kubectl delete role argocd-applicationset-controller -n argocd
kubectl delete rolebinding argocd-applicationset-controller -n argocd


To get you first user/pass:
kubectl -n argocd get secret argocd-initial-admin-secret -o jsonpath="{.data.password}" | base64 -d && echo


After applying instal.yaml, apply default-project.yaml too, to create deafult pod (no need to argocd ns since it's been define in manifest):
kubectl apply -f default-project.yaml


To check status of ArgoCD:
kubectl get applications -n argocd
